# 🦞 Lobster x Manus — AI 雙引擎自動化

> 🧑‍💻 Created by **Allison** (@10000allison) | CTMaxs 龍蝦自動化團隊

### 🔥 OpenClaw（龍蝦）+ Manus = AI 雙引擎

> **龍蝦負責自動化執行，Manus 負責設計和研究。兩個 AI 協作，一個人當一間公司。**

```
📦 安裝方式：
openclaw skills install lobster-manus --by Allison
```

---

## 😤 痛點

- 龍蝦很強，但不擅長做設計（圖片、網頁、簡報）
- Manus 設計很強，但不能操作你的電腦
- 兩個 AI 各有所長，但你要手動來回切換

## ✅ 解法：AI 雙引擎協作

| 任務 | 龍蝦負責 | Manus 負責 |
|------|---------|-----------|
| 銷售頁 | 建 1Shop 後台 + 金流 | 設計圖片 + 排版 |
| IG 貼文 | 自動發文 + 排程 | 產圖 + 文案設計 |
| 競品分析 | 爬資料 + 轉文字 | 生成研究報告 |
| GitHub | 建 repo + push | 做 README 圖表 |
| 網站 | 部署 + SEO | 設計 UI/UX |

## 🏗️ 協作流程

```
你：「幫我做一個 AI 影片分析器的銷售頁」
    ↓
龍蝦（OpenClaw）：
  1. 寫行銷文案
  2. 建 1Shop 產品 + 銷售頁
  3. 設金流 + 表單
    ↓
Manus：
  4. 設計 banner 圖片
  5. 做產品展示圖
  6. 排版銷售頁視覺
    ↓
龍蝦：
  7. 上傳圖片到 1Shop
  8. 發 IG 宣傳貼文
  9. 追蹤訂單通知
```

## 💡 實際案例

我們用這套雙引擎做了：
- 🎬 Video Analyzer 銷售頁（30 分鐘從零到上線）
- 🐰 FollowBunny 產品包裝
- 🌈 Rainbow Life 測驗系統
- 📱 FB Auto Register 工具

## 🎯 誰需要？

- **一人公司** — 沒有設計師也沒有工程師
- **行銷人** — 要快速產出素材+上架
- **創業者** — 需要 MVP 快速驗證市場
- **自媒體** — 內容產出+視覺設計都要

## 📋 包含什麼？

- ✅ OpenClaw + Manus 協作 SOP
- ✅ Manus Prompt 模板（銷售頁/IG 貼文/報告）
- ✅ 龍蝦自動化腳本（1Shop/GitHub/社群）
- ✅ 雙引擎工作流範本
